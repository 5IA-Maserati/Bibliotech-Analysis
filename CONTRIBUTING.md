# Contributing to Bibliotech Analysis

Welcome! This guide helps you contribute in a clear and safe way, even if you are new.

## General Rules
- Work only in your assigned folders.
- Use templates for surveys, requirements, and interviews.
- Write in **English**.
- Follow the Document Style Guide.

## How to Add a New File
1. Copy the correct template from `/templates`.
2. Fill in all required sections.
3. Name the file according to the naming conventions.
4. Save in the correct folder (e.g., `research/surveys/`).

## Commit Guidelines
- Use clear commit messages: `Add survey about library usage`
- Commit often, small changes are better than large ones.

## Security & Quality
- Do **not** upload sensitive personal data from participants.
- Check that all files are properly formatted and complete.
- Ask a team member for review if unsure.

## Questions
- Contact the analysis lead (role/responsible person) for guidance.
